# qingcheng电商
前端采用HTML+Vue，后端采用了SSM+Dubbo+Zookeeper分布式开发方案。主 要包括了后台管理和前台界面，实现了后台商品管理、日志信息、数据分析、定时任务等基本功能。前端实现了统一认证中 心、手机验证码注册、全文检索、商品详情、购物车、微信支付和秒杀等功能
